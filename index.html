Rust to WebAssembly and Biowasm Example

Using a Rust based tool, we compile to WebAssembly in a format that is compativle with Biowasm as an Aioli object. This allows mix and matching of established Biowasm tools with Rust tools. The code powering the example below is as follows: 

   <script  src="https://biowasm.com/cdn/v3/aioli.js"></script>
   <script type="module">

    
    const CLI = await new Aioli([{tool: "biowasmrust",
        program: "fastqe",         
        urlPrefix: "https://wasmodic.github.io/biowasm-rust/",
    }]);

     let output_help = await CLI.exec("biowasmrust --help");
     document.getElementById("output_help").innerHTML = output_help;

     </script>
